# AI Dashboard Platform Product Vision


## Product Definition

We are building an AI Native Dashboard Platform.

The product allows users to create decision-ready dashboards through natural language.

Users describe business goals.

AI generates dashboards.


## Traditional BI vs AI Dashboard


Traditional BI:

User builds dashboard.


AI Dashboard:

User describes intent.

AI generates artifact.


## Core Value

The system acts as:

- BI Consultant
- Data Analyst
- Dashboard Designer
- Frontend Engineer


## User Experience


User:

"帮我生成销售经营驾驶舱"


AI:

Understands business goal

Analyzes dataset

Creates dashboard blueprint

Generates dashboard


User:

"换成苹果风格"

AI:

Creates new artifact version.


## Product Philosophy

Do not optimize for configuration.

Optimize for generation.
